# Bash Scripts Collection

A collection of Bash scripts designed for automation, file management, and other utility tasks.

## Available Scripts

### guessNumber
- **Description**: A number guessing game.
- **Usage**: Run the script and follow the instructions to guess the correct number.

### cruft_remover
- **Description**: Removes unnecessary files from the system or a specific directory.
- **Usage**: Execute the script to clean up unneeded files.

### folder_organiser
- **Description**: Organizes files into folders based on their types.
- **Usage**: Place the script in a directory and run it to categorize files by type.

### gauto
- **Description**: Automates the process of initializing `npm`, `tsc`, and `git`. It also creates a `.gitignore` file, links the local repository with the remote one, adds and commits the changes with a user-provided message, and pushes them to the remote repository. The script runs the necessary commands and asks for user input when needed.
- **Usage**: Use this script to automate the setup and commit process in a Node.js/TypeScript project.

### readLines
- **Description**: Reads a file line by line.
- **Usage**: Run the script with a file as input to process its contents.

### simpleCountdown
- **Description**: A countdown timer.
- **Usage**: Specify the countdown time in seconds and run the script.

### toolkit
- **Description**: A menu item of utility tools cruft_remover and folder_organizer.
- **Usage**: Run the script to access a set of useful tools.

## How to Use
1. Clone the repository.
2. Make the desired script executable.
3. Run the script as described in the usage information.
